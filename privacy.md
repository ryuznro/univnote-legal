 # UnivNote 개인정보 처리방침                                                                             
                                                                                                           
  최종 업데이트: 2026년 4월 25일                         

  ## 1. 수집하는 정보                                                                                      
   
  UnivNote는 사용자의 개인정보를 **수집하지 않습니다**.                                                    
                                                         
  - 회원가입 없음
  - 외부 서버로 데이터 전송 없음
  - 광고/분석 SDK 미사용                                                                                   
   
  ## 2. 데이터 저장 위치                                                                                   
                                                         
  모든 문서, 필기, 이미지는 사용자 기기 또는 사용자 본인의 iCloud 계정에만 저장됩니다.                     
   
  - 로컬 저장: iPad/iPhone 내부 저장소                                                                     
  - iCloud Drive: 사용자가 토글 활성화 시, 본인 Apple ID 의 iCloud 컨테이너
                                                                                                           
  UnivNote 개발자는 이 데이터에 접근할 수 없습니다.
                                                                                                           
  ## 3. 시스템 권한                                                                                        
   
  - **사진 라이브러리** (선택): 노트에 사진 삽입 시 사용. 읽기 전용.                                       
                                                         
  ## 4. 결제 정보

  In-App Purchase (얼리 서포터 팩) 결제는 Apple의 App Store가 처리합니다. UnivNote는 결제 정보(카드 번호   
  등)에 접근할 수 없습니다.
                                                                                                           
  ## 5. 진단 데이터                                      

  iOS 표준 MetricKit으로 수집된 크래시/성능 진단은 사용자 동의 시에만 이메일로 전송됩니다. 자동 전송 없음. 
   
  ## 6. 어린이 보호                                                                                        
                                                         
  UnivNote는 13세 미만 어린이를 대상으로 데이터를 수집하지 않습니다.                                       
   
  ## 7. 변경 사항                                                                                          
                                                         
  본 방침이 변경되면 앱 업데이트와 함께 본 페이지에 게시합니다.                                            
   
  ## 8. 문의                                                                                               
                                                         
  질문 사항: support@univnote.app (출시 후 결정)
